👋 Welcome to LearnHub Backend API

This backend powers the LearnHub Learning Management System (LMS) – a platform designed to deliver modern, role-based online learning experiences.

🔧 Technologies Used

  • Node.js + Express – Backend server and routing
  • MongoDB – NoSQL database for storing users, courses, and content
  • JWT – Authentication
  • Argon2 – Secure password hashing
  • Nodemailer – Email notifications

🚀 Getting Started

cd backend && npm install && npm run dev

Make sure to configure the .env file with your database and email credentials.

🔐 API Routes Overview

  • /api/auth – Register & Login
  • /api/student – Browse courses, enroll, submit assignments
  • /api/instructor – Create courses, add lessons, evaluate submissions
  • /api/admin – Manage users and view dashboard
